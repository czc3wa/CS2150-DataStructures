Prints out the classes in topological order after we have created an Adjacency Matrix.

Returns
There is not return value for this function, we just print out the classes in topological order
Parameters
numClassesThe number of distinct classes that were given in the input file
aMatrixThe Adjacency Matrix that we have computed in the main method
allClassesa vector of string that contains all distinct classes

The most outer for loop makes sure that each distinct class that is given in the output is printed out eventually.

The next two for loops makes sure that we check each column from top to bottom

if we reach a 2 at the top row of a column, we can safely assume that that column has been previously printed out/deleted so we dont have to worry about it anymore

if we ever reach a 1 in a column, we know that the indegree for that column/class is greater than 1 so we can skip it and go to the next one.

if we get to the last row of a column and have only seen 0's, then it is safe to say that the class representing the column has an indegree of 0. we print it out. Then we make sure to delete all the outgoing arrows from that class. also we "delete" the Whole column representing the class by making the first row of that column a 2. Also, we set col equal to numClasses so we can start finding a new class of indegree 0.
